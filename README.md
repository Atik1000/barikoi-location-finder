# barikoi-location-finder

## Project Overview

`barikoi-location-finder` is a Next.js-based location search application built for the Barikoi assignment.
It provides a clean interface to search and explore locations using the Barikoi API.

## Key Features

- Search locations using Barikoi endpoints
- View and explore location results quickly
- Clean UI structure suitable for assignment/demo delivery
- Built with scalable project structure for future enhancements

## Tech Stack

- Next.js
- React
- JavaScript/TypeScript (depending on implementation)
- Barikoi Location API

## Setup and Installation

1. Clone the repository:
	`git clone https://github.com/Atik1000/barikoi-location-finder.git`
2. Move into the project directory:
	`cd barikoi-location-finder`
3. Install dependencies:
	`npm install`
4. Start the development server:
	`npm run dev`
